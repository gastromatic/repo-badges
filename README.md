## Usage

Simply copy the desired Markdown snippet into your own README to embed the badge.

---

**Example:**

```markdown
![Client](https://github.com/gastromatic/repo-badges/raw/main/svg/repo-kind/client.svg)
```

The badge will then be displayed directly in your README.
